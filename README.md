# Islamic Civilizations: Science, Literature & Art

An interactive, single-file HTML presentation on the contributions of Islamic
civilizations to science, literature, and art. Prepared for the Diyanet
"Spirit Program" in Konya.

## Contents
- `index.html` - the presentation deck (open in any modern browser)
- `narration-script.md` - the presenter's full, word-for-word narration script for all 151 slides

## View it live
Once GitHub Pages is enabled, the deck is published at:
`https://<your-username>.github.io/islamic-civilizations/`

## Run locally
Just open `index.html` in your browser. No server or build step is required.
